A CS project on vision-language-action models — policies that map a camera image and a language instruction to robot motion.

This submission combines trained policies (manipulation on the SO-101 arm, driving via a π₀.5 finetune on NVIDIA AV data), published artifacts on Hugging Face, and an interactive site that documents results with logged rollouts and evaluation clips. See the README for the full write-up.
